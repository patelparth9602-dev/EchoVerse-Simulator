# EchoVerse Simulator

A crazy, never-before-seen interactive simulation of a multiversal hive mind! Built with Python, Pygame, and NumPy. Agents (Echoes) swarm towards your mouse, split into parallel universes, evolve traits, and create chaotic "quantum links." Inspired by flocking behaviors but twisted into quantum madness.

## Features
- **Multiverse Splitting**: Press SPACE to duplicate agents into ghostly echo layers.
- **Reality Collapse**: Press R to merge echoes chaotically (with explosion risks!).
- **Evolution**: Press E to mutate traits like speed boosts, glowing auras, or bouncing physics.
- Hypnotic visuals with trails, links, and random "cosmic whispers" in the console.
- Attract the hive with your mouse for emergent swarm behavior.

## How to Run
1. Install dependencies: `pip install pygame numpy`
2. Run the script: `python echoverse.py`
3. Controls:
   - Mouse: Guide the hive.
   - SPACE: Split into multiverses.
   - R: Collapse realities.
   - E: Evolve traits.
   - Close window to quit.

## Requirements
- Python 3.x
- Pygame
- NumPy
